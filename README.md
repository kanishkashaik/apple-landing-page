# apple-landing-page

This is my frontend project where I recreated the Apple website landing page using HTML and CSS.

## 🔥 Features

* Responsive design (mobile friendly)
* Navigation bar with mega menu
* Product showcase sections (MacBook, iPhone, iPad)
* Image slider using pure CSS (radio buttons)
* Interactive buttons and hover effects
* Clean and modern UI

## 🛠️ Technologies Used

* HTML
* CSS
